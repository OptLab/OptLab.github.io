用hexo搭建GitHub博客

Veröffentlicht am 2017-05-23

1. GitHub账号及repository

首先你要有一个GitHub账号，如果没有可以很方便用邮箱注册。有了账号后需要创建一个仓库（repository），将仓库的名字命名为username.github.io这里的’username’一定是GitHub账号的用户名，字母要与用户名大小写相同。
例如，GitHub账户名为OptLab，我们的repository就是OptLab.github.io。

2. 配置本机电脑的安装环境

2.1 安装Node

作用：用来生成静态页面的 到Node.js官网下载相应平台的最新版本，一路安装即可。

2.2 安装Git

作用：把本地的hexo内容提交到github上去。在官网下载，一路安装即可。

3. 正式安装Hexo

Hexo的安装需要在Git Bash（安装Git后在桌面自动生成图标）中进行。

3.1 安装hexo

npm install -g hexo-cli

3.2 创建安装文件夹

创建一个文件夹如c:hexo，并在bash中进入这个文件夹。
cd c:hexo

3.3 初始化博客

hexo init

3.4 配置博客

我们对博客修改相应的配置，我们用到_config.yml文件。

修改网站信息

language和timezone都是有输入规范的，详细可参考语言规范和时区规范。

3.4.1 网址设定

3.4.2 部署配置

3.5 安装server

npm install hexo-server --save
还有一些其他配置，不做似乎也可。

3.5发表一篇文章

hexo new "title"

3.6生成静态页面

hexo generate or hexo g

3.7 启动server

hexo server or hexo s
此时会显示INFO Start processing INFO Hexo is running at http://localhost:4000/. Press Ctrl+C to stop.但是有可能无法打开，这可能与代理服务器配置以及Proxifer的使用有关，建议检验此服务器前关闭代理，关闭proxifer。

3.8 安装deployer

npm install hexo-deployer-git --save
可能会出现一些错误，提示
git config --global user.email "you@example.com"
git config --global user.name "Your Name"

4 获得并添加秘钥

在bash中输入ssh-keygen -t rsa -C "Github的注册邮箱地址"来获取秘钥。
一路Enter过来就好，待秘钥生成完毕，会得到两个文件id_rsa和id_rsa.pub，用带格式的记事本打开id_rsa.pub，Ctrl + a复制里面的所有内容，然后进入https://github.com/settings/ssh： 将复制的内容粘贴到Key的输入框，随便写好Title里面的内容，点击Add SSH key按钮即可。

5 部署

每次部署的步骤，可按以下三步来进行。
hexo clean
hexo generate or hexo g
hexo deploy or hexo d
